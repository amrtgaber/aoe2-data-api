# AoE2 Data Api

## Description

Api for getting the names of all the civs. Will be expanded in future versions. Initially created as a backend api for an AoE2 random civ picker app.

## Docs

Run the project and navigate to `/docs`.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
